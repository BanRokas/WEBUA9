Git ir Github

Git - programa įrenginyje, kuri leidžia programuotojui versijuoti jo rašomą kodą.

Github - platforma (internetinė svetainė), kurioje galima talpinti versijuotą kodą.

Libraries and Frameworks / Bibliotekos ir Karkasai

Bibliotekos - kažkieno kito parašytas kodas (kuris nėra originalioje kalboje), kuriuo galite naudotis savo projektuose.
Norint naudotis bibliotekos kodu, reikią tą biblioteką prijungti prie savo projekto naudojant CDN.
Daugelis CSS bibliotekų veikia tokiu principu, kad bibliotekos viduje yra aprašyta gyva galybė CSS klasių, kuriomis tu naudojiesi rašydamas savo HTML'ą.
Yra išimčių, kai bibliotekos yra ClassLess, kurios neturi CSS klasių ir viską stilizuoja select'indamos tag'us.

Pros / Pliusai

  • Palengvina/pagreitina darbą
  • Yra paruoštų pavyzdžių
  • Nereikia / labai mažai rašyti CSS
  • Įdiegtas responsiveness
  • ...

Cons / Minusai

  • Neunikalus dizainas
  • Reikia išmokti naudotis biblioteka
  • Pasimiršta CSS
  • Sudėtingiau pritaikyti savo stilius (dėl CSS prioritetų)
  • Daug perteklinio kodo
  • Nešvarus HTML'as dėl klasių pertekliaus
  • ...

Preprocessors / Preprocesoriai

CSS su supergaliomis

Suteikia galimybę rašyti CSS kodą naujoviškesniu/mandresniu būdu. Jame lengvai ir patogiai galime kurti kintamuosius, nest'inti CSS kodą, rašyti: sąlygas, funkcijas, ciklus, mix'inus ir t.t.

Naudojant betkurį CSS preprocesorių, reikia parašytą kodą konvertuoti (kompiliuoti) į paprastą CSS'ą, kurį jau prijungsime prie HTML'o. Naršyklė nesupranta perprocesorių, jai reikalingas paprastas CSS'as.

Preprocesorių sąrašas

  • SASS/SCSS
  • LESS
  • Stylus
  • ...
