HTML - HyperText Markup Language


Ant kažkokio teksto

Labas rytas Lietuva! Sekite mūsų naujienas čia su galimybe registruotis naujienlaiškiui!

HTML tag'ai

  • html - grupuoja head ir body tag'us
  • head - meta info apie puslapį
  • body - turinys, kuris atvaizduojamas vartotojui
  • header, nav, main, section, article, div, footer - semantiniai tag'ai
  • meta - meta informacijai nurodyti
  • title - puslapio pavadinimas
  • h1-h6 - heading - antraštė
  • p - paragraph - paragrafas
  • img - image - paveiksliukas
  • i - italic - suteikia tekstui pasvyrimą
  • b - bold - suteikia tekstui paryškinimą
  • span - tuščias linijinis tag'as skirtas apgaubti tekstinį turinį
  • a - anchor - nuoroda
  • button - mygtukas
  • br - line break - eilutės lūžis
  • hr - thematic line break - eilutės lūžis su brūkšneliu
  • ul - unordered list - nerikiuotas sąrašas
  • ol - ordered list - rikiuotas sąrašas
  • li - list item - sąrašo elementas
  • table
  • form

HTML atributai

  • src - source - šaltinis/nuoroda
  • alt - alternative text - tekstas, kuris atvaiduojamas vietoj paveiksliuko
  • width - plotis
  • height - aukštis
  • href - hyperlink reference - nuoroda
  • target - kur bus atidaroma nuoroda

Klaviatūros trumpiniai

  • CTRL A - viską parinkti
  • CTRL ENTER - naujos eilutės sukūrimas
  • ALT TAB - vaikščiojimas tarp atidarytų programų
  • CTRL Z - atšaukia paskutinį veiksmą
  • CTRL Y - atvirkščias CTRL Z
  • CTRL F - atidaromas paieškos laukas
  • CTRL D - multi selektina parinkties duplikatus
  • ALT mouseClick - sukuria papildomus kursorius
  • CTRL / - parinktų eilučių komentavimas
  • CTRL T - sukuriamas naujas tab'as (svetainėje)
  • ALT Z - toggle'ina word wrap
  • SHIFT ALT DOWN/UP - parinktų eilučių duplikavimas žemyn/aukštyn
  • SHIFT ALT F - automatinis tabuliavimas
