CSS Stilių pritaikymo taisyklės

Naršyklės numatytieji stiliai

Didžioji dalis tag'ų turi kažkokius tais numatytuosius naršyklės stilius, kurie gali minimalistiškai skirtis tarp skirtingų naršyklių.

Normalize CSS - normalizuoja tuos galimai minimalistiškus stilių skirtumus tarp naršyklių ir panaikina/sugrąžina nenorimus stilius į pradines būsenas.

Reset CSS - panaikina visus numatytuosius stilius visiems elementams.

Tėviniai ir vaikiniai elementai

Tėvinis elementas GALI turėti daugelį vaikinių elementų.

Vaikinis elementas TURI TIK vieną tėvinį elementą.

Paveldėjimas

Vaikinis elementas gali paveldėti stilius (jeigu šie yra paveldimi) iš tėvinio elemento.

CSS stilių prioritetai

  1. !important - siūloma nenaudoti
  2. Inline style - linijinis stilius
  3. Selector specificity - Selektoriaus specifikacija
    1. 100 - id
    2. 010 - class
    3. 001 - tag
  4. Order - Eiliškumas
  5. User agent stylesheet - Naršyklės numatytieji stiliai
  6. Inheritance - Paveldėjimas

Dėžutės Modelis

  1. Content - turinys - centrinė dalis
  2. Padding - pamušalas - tarpelis tarp turinio ir rėmelio
  3. Border - rėmelis - linija tarp Padding ir Margin
  4. Margin - paraštė - atstumas tarp elemento ir jo aplinkinių elementų
  5. Positions
