Lentelė

  • table - pagrindinis lentelės tag'as
  • thead - stulpelių pavadinimų laukas
  • tbody - laukas atvaizduoti duomenims
  • tfoot - išvestinė
  • tr - table row - lentelės eilutė
  • th - table heading - stulpelių pavadinimai
  • td - table data - duomens laukas
  • caption - lentelės pavadinimas

Lentelės skirtos atvaizduoti kažkokiai informacijai.
Lentelės susideda iš eilučių, kurių viduje kuriami elementai sukuria stulpelius.

CSS Nest'inimą

CSS selektorius galima nest'inti. Vieno selektoriaus viduje, galime rašyti kitą selektorių, šitaip išvengdami tų pačių selektorių kartojimo.

Pseudo Klasių selektoriai

Dalis pseudo klasių reaguoja į kažkokius tais vartotojo veiksmus.
Dalis pseudo klasių gali kažkaip kitaip stilizuoti elementus priklausomai nuo jų pozicijos.

  • Nuorodos (a - anchor)
    1. :link - nelankyta nauja nuoroda
    2. :visited - lankyta nuoroda
    3. :hover - reaguoja į pelės užvedimą ant elemento (NE TIK su nuorodomis)
    4. :active - reaguoja į paspaudimo momentą / aktyvią navigacinę nuorodą
  • :first-child
  • :last-child
  • :first-of-type
  • :last-of-type
  • :nth-child(n)
  • :nth-last-child(n) - tas pats kas nth-child(n), tik iš kito galo
  • :nth-of-type(n)
  • :nth-last-of-type(n)
  • Nebūtina:
    • :root
    • :not(selector)
    • :empty
    • :fullscreen
    • :lang(language)
    • :target
    • :is()
    • :has()
    • :where()
  • ...

  • ::before - sukuria turinį prieš elementą
  • ::after - sukuria turinį po elemento
