Kas yra Responsive Design

Dizainas, kuris keičiasi priklausomai nuo vartotojo ekrano dydžio.

Breakpoints / lūžio taškai

Lūžio taškas - tai kažkoks nustatytas ekrano plotis (skaičiuojamas px arba em), kurį pasiekus yra pradedamas taikyti kitas dizainas (įsigalioja kiti CSS stiliai / kitas mediaQuery).

Mobile First

Mobile First - pradžioj mobilusis - dizaino principas reiškia, kad pradedame stilizuoti svetainę ją pritaikydami mobiliesiems įrenginiams. Ir ekranui plečiantis įsigalioja kiti stiliai, kurie pritaiko dizainą platesniems ekranams.

Taip yra daroma dėl įvairių priežasčių

  • Daugiau vartotojų lanko internetines svetaines naudodami telefoninius įrenginius
  • Turint mažiau vietos esi verčiamas pateikti glaustesnę informaciją
  • Lengviau yra pridėti papildomo turinio, negu atimti esamą.
  • ...
