CSS Pagrindai

CSS - Cascading Style Sheets - pakopiniai stiliaus šablonai yra naudojami norint stilizuoti HTML elementus.

CSS rašymo būdai

  • Inline - linijinis - tag'ui priskiriame atributą style ir jo reikšmės viduje rašome stilius.
  • Internal - vidinis - CSS rašomas HTML faile, style tag'o viduje.
  • External - išorinis - CSS rašomas atskirame faile, kurį reikia prijungti prie HTML.

CSS Selektoriai

CSS selektorius - būdas parinkti kokiam HTML elementui norime priskirti toliau rašomus stilius.

Selektorių sąrašas

  • tag - p/h1/section/... - yra parenkami visi nurodyti tag'ai
  • * - visko selektorius - parenka visus elementus
  • .klasesVardas - klases selektorius - parenka elementus, kuriems yra priskirtas nurodytas klasesVardas
  • #idVardas - id selektorius - parenka elementą, kuriam yra priskirtas nurodytas idVardas
  • ...

Spalvos

  • Spalvų pavadinimai (tik apie 140)
  • RGB - red green blue, nuo 0 iki 255
  • RGBA - red green blue ir alpha, alpha - nuo 0 iki 1, 0.001 žingsniu - skirtas nurodyti spalvos permatomumą
  • HEX - hexdecimal - šešioliktainė nuo 0 iki f - trims RGB spalvoms
  • HSL - hue saturation lightness

Fonai

Display

  • Inline - linijiniai. Tokie elementai, kurie užima TIK tiek pločio kiek reikalauja jų turinys, todėl gali atsistoti vienas šalia kito. Negalima suteikti pločio arba aukščio nustatymų.
  • Block - blokiniai. Tokie elementai, kurie užima 100% tėvinio elemento pločio užtai negali atsistoti vienas šalia kito. Galima suteikti pločio ir aukščio nustatymus.
  • Inline-Block - linijiniai-blokiniai. Paima geriausias dalis iš inline ir block elementų - gali atsistoti vienas šalia kito, nes originaliai užima tik tiek vietos, kiek reikalauja jų viduje esantis turinys. Taip pat gali turėti pločio ir aukščio nustatymus.
