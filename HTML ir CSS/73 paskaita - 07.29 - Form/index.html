Form

Formos naudojamos norint atvaizduoti vartotojams lauką, skirtą suvesti kažkokiai informacijai (prisijungimas, registracija, žinutės siuntimas, bilietų pirkimas...), kuri po to gali būti pateikiama ir pagal įvestus duomenis - sugeneruojamas atsakymas.

Formos tag'ai

  • form
  • label
  • input - turi DAUG variacijų
  • textarea
  • select
  • option
  • datalist
  • optgroup
  • output
  • fieldset
  • legend

Formos tag'ų atributai

  • type - nustato kokio tipo bus įvestis
    • text - teksto įvestis
    • number - skaičiaus įvestis
    • range - skaičiaus įvestis (slider)
    • password - slaptažodžio įvestis
    • email - el pašto įvestis
    • url - nuorodos įvestis
    • file - failo įkelimo įvestis
    • datetime-local - data ir laikas
    • color - spalvos pasirinkimas
    • radio - vienas iš pasirinkimų
    • checkbox - keletas iš pasirinkimų
    • ...
  • for - apjungia label su input
  • id - apjungia input su label
  • name - suteikia įvesčiai arba įvesčių grupei pavadinimą
  • placeholder - parodomoji įvestis
  • minlength - minimalus įvesties ilgis
  • maxlength - maksimalus įvesties ilgis
  • required - privalomas įvesties laukas
  • selected - iš anksto parinkta opcija
  • disabled - neleistina pasirinkti opcija
  • min - minimalus skaičius/data
  • max - maksimalus skaičius/data
  • step - skaičiaus žingsnis (kiek už kablelio)
  • accept - kokio tipo failus rodys atidarytame lange
