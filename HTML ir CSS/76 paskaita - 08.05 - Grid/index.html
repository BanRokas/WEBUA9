Grid

Grid atvaizdavimo sistema leidžia mums sukurti norimo tankumo tinklelį kokiame nors konteineryje (elemente) ir tuomet to tinklelio viduje išdėlioti to konteinerio elementus į atitinkamas vietas.
Sukuriame grid'ą nurodydami stulpelių ir/arba eilučių dydžius bei kiekius. Tuomet grid'o viduje sustumdai vaikinius elementus į norimas vietas nusakydamas ties kuria tinkelio vieta elementas turi prasidėti ir ties kuria vieta turi pasibaigti.
Grid kaip ir Flex turi daug stilių, kurie skirti tiek tėviniui (konteineriui) elementui, tiek vaikiniams elementams. Taip pat Grid turi naujų matavimo vienetų ir pagalbinių funkcijų.

Grid savybės

Tėviniai savybės

  • display: grid;
  • grid-template-columns
  • grid-template-rows
  • grid-template: grid-template-rows / grid-template-columns;
  • grid-template-areas
  • row-gap
  • column-gap
  • gap: row-gap column-gap;
  • grid-auto-flow: row (default) | column | row dense | column dense;
  • align-items
  • justify-items
  • place-items: align-items justify-items;
  • align-content
  • justify-content
  • place-content: align-content justify-content;
  • grid-auto-rows
  • grid-auto-columns
  • grid: grid-template-rows / grid-template-columns / grid-template-areas / grid-auto-rows / grid-auto-columns / grid-auto-flow;

Vaikiniai savybės

  • grid-row-start
  • grid-column-start
  • grid-row-end
  • grid-column-end
  • grid-row: start / end;
  • grid-column: start / end;
  • grid-area: name | grid-row-start / grid-column-start / grid-row-end / grid-column-end;
  • align-self
  • justify-self
  • place-self: align-self justify-self;

Matavimo vienetai bei funkcijos

  • fr - fraction
  • repeat(kiek, ką)
  • span - per kelis stulpelius/eilutes tęsiasi
  • min-content
  • max-content
  • auto
  • fit-content()
  • minmax()
  • min()
  • max()
  • masonry()
  • subgrid()

Grid VS Flex

Tiek Grid tiek Flex turinio išdėstymo nustatymai yra labai galingi ir turi įvairių panaudojimo atvejų. Bet vienoje situacijoje reikia naudoti tik vieną iš jų. Kada naudoti Grid, o kada Flex?
Flex dažniausiai yra naudojamas kai jo viduje esantys elementai turi būti išdėstyti vienoje dimensijoje (galbūt su wrap'inimu), o Grid naudojamas kai jo viduje esantys elementai turi būti išdėstyti keliose dimensijose.
